# Pantheon+ ΛCDM Cosmology Analysis

## Overview
This project analyzes Type Ia supernova data from the Pantheon+SH0ES dataset to study cosmic expansion and estimate key cosmological parameters.

## Objectives
- Estimate the Hubble constant (H₀)
- Compute the age of the Universe
- Compare low- and high-redshift supernova samples

## Methods
- ΛCDM cosmological modeling
- Curve fitting and parameter estimation
- Residual analysis

## Tools Used
- Python (NumPy, SciPy, Matplotlib)
- Data analysis techniques

## Results
- Estimated H₀: 73.53 ± 0.17 km/s/Mpc
- Age of Universe: 12.36 Gyr

## Data Source
Pantheon+SH0ES Supernova Dataset
